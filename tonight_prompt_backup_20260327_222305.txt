야간 자동 작업: CEO 리뷰 → 기능 구현 → 테스트 → 코드리뷰 파이프라인

## 규칙
- 주석/커밋 메시지 한국어
- 기존 코드 패턴 따르기
- 각 작업 완료마다 커밋
- TASK.md 갱신 필수
- 각 기능마다 4단계 파이프라인 필수: CEO리뷰 → 구현 → 테스트 → 코드리뷰

---

## 기능 A: 4-3 — 블록체인 생태계 #1: 비트코인 편

### Phase 1: CEO 리뷰
- **문제**: 블록체인 생태계 #1: 비트코인 편
- **스코프**: 관련 파일/모듈 확인 필요
- **판정**: 야간작업에서 판단

### Phase 2: 구현
TASK.md의 4-3 항목 기반으로 구현. 기존 코드 패턴 따르기.

### Phase 3: 테스트
관련 테스트 파일 확인 후 테스트 추가/실행.

### Phase 4: 코드리뷰
**Pass 1 CRITICAL**: 보안, 예외 처리
**Pass 2 INFO**: 코드 스타일, 로깅
