# IV 시그마 분석 스킬 (sigma-analysis)

> **트리거**: 아래 표현이 나오면 이 스킬을 즉시 실행한다.
> - "시그마 분석" / "IV 분석" / "sigma"
> - "변동성 분석" / "{티커} 시그마"
> - 종목 리서치 시 변동성 확인 필요 시

---

## 목적

개별 종목의 내재변동성(IV) 또는 역사적변동성(HV)을 기준으로 현재 주가가 통계적으로 어디에 위치하는지(시그마 수준)를 분석한다.

참조: `scripts/iv_sigma_report.py` (`analyze_ticker` 함수)

---

## 실행 순서 (순서 준수 필수)

### STEP 1 — 실행 시간대 확인

| 시간대 | IV 가용성 | 사용할 변동성 |
|--------|----------|-------------|
| 미국 장중 (9:30~16:00 ET) | ✅ 가용 | **IV** (내재변동성) |
| 미국 장외 / 한국 시간 | ❌ 미제공 | **HV20** (20일 역사적변동성) 폴백 |

HV20 폴백은 정상 동작이다 — 경고 표시하되 분석은 유효.

### STEP 2 — 분석 실행

```bash
cd /home/window11/stock
source .venv-wsl/bin/activate

# 단일 종목
python scripts/iv_sigma_report.py TSLA

# 복수 종목
python scripts/iv_sigma_report.py TSLA PLTR NVDA
```

### STEP 3 — 결과 해석

`analyze_ticker` 출력에서 핵심 지표:

| 지표 | 설명 | 판단 기준 |
|------|------|----------|
| 현재 IV (또는 HV20) | 연간 변동성 % | — |
| IV Percentile | 과거 1년 대비 위치 | >80 = 고변동, <20 = 저변동 |
| 1σ 범위 | 68% 확률 범위 | 현재 주가가 범위 내/외 |
| 2σ 범위 | 95% 확률 범위 | — |
| 시그마 수준 | 현재 가격의 σ 위치 | |1σ|<1 = 정상, >2 = 극단 |

### STEP 4 — 결과 보고

```
📊 시그마 분석 — {날짜} {장중/장외}

| 종목 | 가격 | 변동성 | IV%ile | 1σ 범위 | 현재 σ |
|------|------|--------|--------|---------|--------|
| TSLA | $180 | IV 52% | 65 | $165~$195 | +0.3σ |
| PLTR | $85 | HV20 48% | 72 | $77~$93 | -0.2σ |

⚠️ 장외시간 — HV20 폴백 사용 (IV 미제공)
```

### STEP 5 — 해석 코멘트

시그마 수준에 따른 해석:

| σ 수준 | 해석 |
|--------|------|
| -2σ 이하 | 통계적 저평가 영역 (극단적 하락) |
| -1σ ~ -2σ | 평균 이하, 관심 영역 |
| -1σ ~ +1σ | 정상 범위 |
| +1σ ~ +2σ | 평균 이상, 과열 주의 |
| +2σ 이상 | 통계적 고평가 영역 (극단적 상승) |

## 주의사항

- 시그마 분석은 **통계적 참고 도구**이지 투자 시그널이 아님
- IV와 HV20은 성격이 다름 — IV는 시장 기대, HV는 과거 실현. 비교 시 명시
- 시그마 분석 결과를 아티클에 인용할 때는 출처와 시점 명기
- 실행 시 yfinance 의존 — 네트워크 오류 시 재시도
