# Skills

반복적인 작업을 스킬로 정의하여 재사용합니다.

## 작성 방법

각 스킬은 독립된 `.md` 파일로 작성:

```
skills/
  dashboard-export.md    ← 대시보드 엑셀 내보내기
  data-collection.md     ← 데이터 수집 실행
  test-run.md            ← 테스트 실행 + 커버리지 확인
```

## 스킬 파일 구조

```markdown
# {스킬명}

## 목적
- 이 스킬이 해결하는 문제

## 실행 조건
- 언제 이 스킬을 사용하는가

## 절차
1. 단계별 실행 방법
2. ...

## 주의사항
- 주의할 점
```
