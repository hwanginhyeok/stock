# 아웃라인 — 029 GEOPO: 세계에서 가장 좁은 곳의 전쟁

> 상태: 아웃라인 v1 | 작성자: AI | 날짜: 2026-03-31

---

## 핵심 질문

왜 폭 33km의 해협 하나가 세계 경제의 목을 쥘 수 있는가?

---

## 논리 흐름

```
통념: "중동 전쟁은 이란-이스라엘의 종교/정치 갈등이다"
  ↓
분해: 에너지 병목(호르무즈) + 통상 병목(수에즈)의 지배권 싸움
  ↓
Gap: IMEC는 병목 우회 시도이나, 전쟁이 필요성을 증명하면서 동시에 불가능하게 만든다
  ↓
전개: 각국의 대응은 병목 의존도와 정확히 비례한다
  ↓
결론: 세계 경제의 가장 취약한 곳이 가장 불안정한 곳과 겹친다
```

## 섹션 구조

### §0 도입 — 세계에서 가장 좁은 곳 (~25줄)
- 핵심 포인트: 폭 33km, 수심 60m, 항행 가능 폭 6km — 세계 석유의 20%가 2차선 도로를 지남
- 사용할 데이터: 일일 2,000만 bpd (S03, M01)
- 오프닝 장면: 2026.03.04, IRGC가 VHF 무선으로 해협 폐쇄 선언. 그 순간부터.
- 유가 $71 → $166 (30일 134% 상승)
- 핵심 질문 제시

### §1 병목의 해부학 — 호르무즈 해협 (~50줄)
- 핵심 포인트: 이 해협이 "왜" 대체불가능한가를 제1원칙으로 분해
- 지리적 구조: 33km 폭, 편도 2레인, 이란과 오만 사이
- 통과 물량: 20M bpd 원유 + 글로벌 LNG 25%
- 봉쇄 4종 세트: (1) 미사일/드론/보트 물리적 공격, (2) 기뢰 부설, (3) 위성 스푸핑/GNSS 재밍, (4) 보험료 폭등 유발
- 이란의 "선별적 봉쇄": 중국·러시아·인도 통과 허용 / 미국 동맹 차단 → 봉쇄를 외교 도구로 전환
- 대안 파이프라인의 한계: 사우디(7M) + UAE(1.8M) + 이라크(1.6M) = 현실 2.6~5.5M bpd → 정상의 13~28%만 커버
- 경제적 충격: 두바이유 $166, 유럽 가스 2배, 비료 +50%, IEA 4억 배럴 방출, 1970년대 이후 최대 에너지 교란
- 📎 시각화 1: 호르무즈 병목 수치 인포그래픽

### §2 타임라인 — 어떻게 여기까지 왔나 (~55줄)
- 핵심 포인트: 2024-2026, 3년 간의 에스컬레이션. 보복의 보복이 전면전으로.
- **2024 — 직접 교전의 시작**
  - 4월: 다마스쿠스 이란 대사관 공습 → True Promise I (드론+미사일+탄도 이스라엘 직접 타격)
  - 7~9월: 하니예 테헤란 암살, 나스랄라+IRGC 사령관 레바논 암살
  - 10월: True Promise II (180발 탄도) → Days of Repentance (F-35 100기, 20지점 3파 공습)
- **2025 — 12일 전쟁 + 핵 위기**
  - 봄: 미-이란 핵 협상 5차 → 실질 진전 없음 (미국: 전면 해체 요구 / 이란: 핵만 협상)
  - 6월: IAEA NPT 위반 선언 → 이스라엘 기습 → True Promise III (550발+1,000드론) → 미국 참전 (벙커버스터 핵시설 3곳) → 6/24 휴전
  - 이후: JCPOA 탈퇴, IAEA 사찰 중단, 60% HEU ~440kg 보유 (핵무기 ~10개 분량)
- **2026 — 전면전**
  - 2/27: 오만 중재 "돌파구" 발표 → 3/2 협상 재개 예정
  - 2/28: Operation Epic Fury — 기습공격, 12시간 900회 타격, 하메네이 사망
  - True Promise IV — 500발+ 미사일 + 2,000대 드론 → 12개국 타격
  - 3/4: 호르무즈 봉쇄 선언
  - 3/28: 후티 참전 — 이스라엘 탄도미사일 + 홍해 재봉쇄 위협
- 비유: "보복의 보복" — 매번 더 커지는 True Promise (I→II→III→IV)
- 📎 시각화 2: 에스컬레이션 타임라인 + 유가 곡선

### §3 이란이 쏜 곳들 — 12개국 타격 지도 (~45줄)
- 핵심 포인트: True Promise IV — 역사상 최대 규모의 미사일/드론 동시 공격
- 타격 규모: 500발+ 탄도미사일 + 2,000대 드론 (40% 이스라엘, 60% 역내 미군기지)
- 국가별 피해 (테이블):
  - 이스라엘: 도시+군사시설 (1차 타겟)
  - UAE: 357 탄도 + 1,806 드론 + 15 순항 → 군 2명 + 민간 6명 사망 (IMEC 허브)
  - 바레인: 미군기지 + 알루미늄 시설, 3명 사망
  - 쿠웨이트: 미군기지 + Port Shuaiba, 군 4명 + 민간 4명 사망
  - 카타르: LNG 시설 (생산 중단) + 미군기지
  - 사우디: 아람코 정유 + THAAD 기지, 2명 사망 (IMEC 경유국)
  - 오만: 항만 연료저장, 3명 사망
  - 요르단: 미군기지 + THAAD 기지 (IMEC 경유국)
  - 이라크 쿠르디스탄: 에르빌 공항 + 미 영사관, 450발+ 드론/미사일
  - 아제르바이잔: 나히체반 공항, 학교 인근 (이란 부인)
  - 터키: 상공에서 NATO 방공이 요격 (비의도적)
  - 키프로스: 영국 아크로티리 기지
- 전체 사상자: 2,300명+ (Al Jazeera)
- 제1원칙 질문: "왜 12개국을 동시에?" → 미군기지가 있는 곳 = 타격 대상. 이란의 비대칭 전략: 해협 봉쇄(방패) + 미사일 살포(창)
- 📎 시각화 3: 이란 타격 12개국 테이블 (국가·무기·피해·IMEC 관련성)

### §4 누가 움직이는가 — 각국의 대응 (~55줄)
- 핵심 포인트: 대응의 성격은 호르무즈 의존도의 함수다
- **교전국**
  - 미국: 50,000명 (2003년 이후 최대), 항모 2척 (Lincoln, Ford), F-35C, 해병원정대. 3/19 해협 개방 작전
  - 이스라엘: 공습 주도 + 헤즈볼라 250회 공습, 800명 사살
- **에너지 수호자**
  - 인도: Operation Urja Suraksha — 구축함 5척, 선박 22척 호송 (20척 LNG/LPG/원유). 중동 원유 의존도 80% → 해군 직접 투입
  - 파키스탄: Operation Muhafiz-ul-Bahr — 해안 방어 + 상선 호송. 동시에 4국 중재 (터키·이집트·사우디). "중립 + 중재 + 보호" 세 역할 동시 수행
- **해상 안보 참여자**
  - 프랑스: 유럽 주도 호송 (Operation Aspides), 프리깃 2척. 그리스·이탈리아·네덜란드 참여
  - 22개국 공동성명 (3/19~21): 해협 개방 요구
- **특혜 수혜자**
  - 중국: 이란 최대 고객 → 최초 통과 허용. "전쟁은 시작되지 말았어야" — 비판하면서 수혜
  - 러시아: 통과 허용 + 외교적 해결 촉구
- **중재자**
  - 파키스탄: 이슬라마바드를 협상 장소로 제안 (3/23), 4국 외무장관 회의 (3/29)
  - 터키: 미국 군사행동 반대, 중재 제안
- UN 안보리 2817호: 13-0-2 (중·러 기권). 135개국 공동 제안.
- 제1원칙: 인도(80% 의존) → 군함. 중국(최대 고객) → 특혜. 유럽(LNG) → 호송. 의존도 = 대응 강도.

### §5 IMEC — 병목을 우회하려는 길 (~30줄)
- 핵심 포인트: IMEC의 역설 — 가장 필요해진 순간에 가장 불가능해졌다
- IMEC 경로: 인도 → UAE → 사우디 → 요르단 → 이스라엘 → 유럽
- 목적: 중국 BRI 대안 + 에너지·데이터·무역 통합 통로
- 진행 경과: 2023 G20 발표 → 2025.04 착공 → 인도-UAE IGFA 체결
- 전쟁의 영향: 경유국 3곳(UAE·사우디·이스라엘) 전부 피격 → 사실상 동결
- 역설: 호르무즈 봉쇄 + 수에즈 위협(후티)이 IMEC의 존재 이유를 가장 명확히 증명
- "리셋 단계" — 지역 안정이 IMEC의 전제조건. 전쟁이 끝나야 길이 열린다

### §6 결론 — 세계에서 가장 좁은 곳 (~20줄)
- 호르무즈 = 에너지의 목. 수에즈 = 통상의 목. IMEC = 목을 우회하려는 길
- 세계 경제의 가장 취약한 곳이 가장 불안정한 곳과 겹친다
- 각국의 대응은 이 취약점에 대한 노출도의 함수
- 결론: 지도에 줄을 그어보라. 에너지가 지나는 곳, 미사일이 날아간 곳, IMEC가 지나려는 곳 — 세 줄이 겹치는 곳이 지금 세계에서 가장 중요한 곳이다
- 독자에게 질문: "당신의 포트폴리오는 이 해협에 얼마나 노출되어 있는가?"
