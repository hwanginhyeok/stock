# 달 반도체 공장 — 제1원칙으로 분해하면

> 수집일: 2026-03-02
> 용도: 아티클 002 (우주편) 리서치 노트 3/3
> 범위: 머스크의 Dirty Fab 테제, 팹 제1원칙 분해, 달 물리적 이점, 운송 계산, 타임라인

## 1. 출발점 — 머스크의 "Dirty Fab" 테제

### 원문 (Moonshots Podcast #220, 2025-12-22, Giga Texas)

> "I think they are getting clean rooms wrong in these modern fabs."
>
> "I am going to make a bet here, that Tesla will have a 2nm fab,
> and I can eat a cheeseburger and smoke a cigar in the fab."

### 핵심 논리

1. 웨이퍼는 이미 FOUP(Front Opening Unified Pod)에서 질소 밀봉 상태로 이동한다
2. 공정 장비 안에서도 밀봉된 미니 환경에 있다
3. 그런데 왜 **건물 전체**를 클린룸으로 만드는가?
4. 웨이퍼를 격리하면 되지, 건물을 격리할 필요가 없다
5. 인간을 빼면 → 클린룸이 필요 없다 → 팹이 극적으로 줄어든다

### 이것을 달에 적용하면?

인간 대신 **로봇**, 클린룸 대신 **공짜 진공**.
"팹을 줄이는 것"이 아니라, **팹의 전제 자체가 바뀐다**.

## 2. 제1원칙 분해 — "팹은 왜 이렇게 큰가?"

### 오염원 분석

| 오염원 | 기여도 |
|--------|--------|
| **인간** (피부, 호흡, 움직임) | **75~80%** |
| 장비·공정 | 15~20% |
| 클린룸 인프라 자체 | 2~5% |
| 소모재 | ~5% |

**출처**: Cleanroom Solutions, Precision Companies, Prudential Uniforms

인간은 분당 **10만~100만 개** 입자를 방출한다 (걸을 때).
하루에 피부 세포 **10억 개**를 떨군다.
최고급 방진복을 입어도 완전 차단은 불가능하다.

클린룸 전용 로봇의 입자 방출: **거의 0개** (ISO 14644 인증).

### 팹 건물 구조 — 인간 때문에 큰 부분

| 층 | 용도 | 전체 대비 | 인간 제거 시 |
|----|------|-----------|-------------|
| 팬 덱 (4층) | 공기순환·HEPA 필터 | ~25% | **제거** |
| 클린룸 (3층) | 생산 (가운룸, 복도 포함) | ~25% | 가운룸·복도·에어락 **제거** |
| 서브팹 (2층) | 펌프, 가스, 화학물질 | ~30% | 유지 (공정 장비 지원) |
| 유틸리티 (1층) | 전기, 냉각, HVAC | ~20% | HVAC 부분 **제거** |

**TSMC Fab 18**: 총 95만 m², 클린룸 16만 m². 클린룸은 전체의 **17%**.

### 전력 분배

100MW 팹 기준:

| 용도 | 전력 | 비중 | 달에서 |
|------|------|------|--------|
| HVAC / 공기순환 | 30~50 MW | 30~50% | **제거** |
| 냉각 (칠러) | 10~20 MW | 10~20% | **대부분 제거** (우주 복사 방열) |
| **공정 장비** | **40~50 MW** | **40~50%** | **유지** |
| 조명, 사무실, 인간 시설 | ~3 MW | ~3% | **제거** |
| UPW, 가스, 유틸리티 | 5~10 MW | 5~10% | 부분 축소 |

**인간을 빼면 전력의 40~60%가 사라진다.**

### 머스크의 논리 검증 — 맞는 부분과 과장된 부분

**맞는 부분:**
- 오염의 75~80%가 인간 → ✅ 데이터로 검증됨
- 웨이퍼는 이미 FOUP에서 이동 → ✅ 사실
- 클린룸 로봇은 입자 거의 0 → ✅ 사실
- 업계가 Lights-Out 방향으로 이동 중 → ✅ 사실 (Gartner: 2035년 예측)

**과장된 부분 (지구에서):**
- "치즈버거 먹으며 팹 운영" → ❌ 공정 장비 내부에서 웨이퍼가 노출되는 순간이 있음
- 팹 크기 극적 축소 → ⚠️ 인간 관련 공간은 5~15%에 불과 (지구에서)
- HVAC 제거 → ❌ 온도 안정성(0.1°C)은 공정 요구사항, 인간 편의가 아님

**그런데 — 달에서는 다르다.**

## 3. 달에서는 방정식이 바뀐다

### 진공 비교

| 환경 | 분자 밀도 (/m³) |
|------|-----------------|
| 지구 대기 | ~10²⁵ |
| ISO 1 클린룸 (최고 등급) | ~10²⁵ (가스 분자), ≤10개 입자(≥0.1μm) |
| **달 표면 진공** | **~10¹⁰~10¹²** |
| **비교** | **10¹³배 (10조 배) 깨끗** |

달의 진공은 분자 밀도 기준으로 지구 최고의 클린룸보다 **10조 배** 깨끗하다.
이 환경이 **공짜**로 존재한다. HVAC도, HEPA 필터도, 팬 덱도, 방진복도 필요 없다.

### 진동 — 리소그래피의 최적 환경

EUV 리소그래피는 **10nm RMS 이하** 진동을 요구한다.

| 항목 | 지구 | 달 |
|------|------|-----|
| 지진 에너지 | 기준 | **지구의 1/80** |
| 판구조 운동 | 있음 | **없음** |
| 교통/산업 진동 | 상시 | **없음** |
| 바람 | 있음 | **없음** |
| 대응 비용 | 공기 베어링, 특수 기초 (수억 $) | **불필요** |

**리소그래피를 위한 가장 조용한 환경이 달에 있다.**

### 크기 축소 — 1/10 ~ 1/100

지구에서 인간 관련 시설 제거: **5~15%** 축소.
달에서 인간 + 대기 + HVAC 제거: **80~90%** 축소.

| 제거 항목 | 절감 |
|-----------|------|
| 팬 덱 / 공기순환 층 전체 | ~25% 건물 |
| HVAC 유틸리티 | ~15% 건물 |
| 가운룸, 복도, 에어락, 사무실 | ~10% 클린룸 |
| 클린룸 외벽 / HEPA 패널 전체 | 제거 |
| 소방, 비상구, 엘리베이터, 조명 | 제거 |

**결과:**
- 동일 규모 1/10: **~5,000~15,000 m²** (vs TSMC 95만 m²)
- 저량 생산 1/100: **~500~2,000 m²**

### 전력 — 현실적 범위

| 시나리오 | 장비 수 | 총 전력 | 달에서 가능? |
|----------|---------|---------|-------------|
| 1/10 규모 | ~120개 (EUV 포함) | **~6 MW** | ✅ 핵분열 원자로 |
| 1/100 규모 | ~20개 (DUV) | **~1 MW** | ✅ 킬로파워 다수 or 태양광 |
| EUV 1대 + 최소 공정 | 최소 | **~3 MW** | ✅ 핵분열 원자로 |

EUV 1대 = ~1MW. DUV(28nm)로 가면 ~100kW (1/10).
우주용 특수 칩 (방사선 차폐, 센서, MEMS)은 **DUV로 충분**.

## 4. 먼지 — 달의 유일한 적

### 레골리스의 특성

| 특성 | 설명 |
|------|------|
| 형태 | 날카로운 유리질 파편 (풍화 없음, 각진 상태 유지) |
| 크기 | 평균 70μm, 미세 입자 20% 이상 (<20μm) |
| 정전기 | 태양 UV/X선으로 대전됨 — 표면에 강하게 부착 |
| 부양 | 미세 입자(<1μm)가 표면에서 **~1m** 높이까지 정전기 부양 |
| 건강 위험 | 해리슨 슈미트 (아폴로 17): 달 먼지 알레르기 반응 |

### 해결: 밀봉된 진공 챔버

지구 팹의 먼지 방어: 공기를 필터링한다 (공기가 먼지를 운반).
달의 먼지 방어: **밀봉된 진공 챔버**면 된다.

| 접근 | 설명 |
|------|------|
| **가압 불필요** | 팹은 진공 챔버 — 대기가 없으므로 먼지가 떠다니지 않음 |
| **정전기 부양 회피** | 구조물을 **10m 이상** 올리면 부양 먼지 영역 회피 |
| **재료 반입** | 밀봉 컨테이너 도킹 → 외부 표면은 내부에 노출되지 않음 |
| **EDS (정전기 먼지 방패)** | NASA 시연: **99% 먼지 제거율**, TRL 7 |
| **액체질소 세척** | **98.4% 질량 제거**, 10μm 이하 **95.9%** 제거 |

**핵심 인사이트**: 지구에서는 공기가 먼지를 운반한다. 달에서는 공기가 없다.
먼지는 직접 접촉이나 정전기로만 이동한다 — **밀봉으로 해결 가능한 문제**.

- **Gartner 예측**: 완전 자동화 팹 **2035년** 주류화

## 6. 실현 불가능한 것 — 솔직한 평가

### 달에서 반도체 "칩"을 만드는 건 아직 비현실적인 이유

| 항목 | 요구 | 달 현실 | Gap |
|------|------|---------|-----|
| 실리콘 순도 (IC 칩) | **9N** (99.9999999%) | MRE 추출 **5N** | 4자릿수 |
| 초순수 (UPW) | 대량 필요 | 극지 얼음 → 정제 필요 | ISRU 미성숙 |
| 특수 가스 (SiH₄, WF₆) | 지속 공급 | 지구에서 수송 only | 물류 의존 |
| 포토레지스트 | 지속 공급 | 지구에서 수송 only | 물류 의존 |

### 그러나 — 단계적으로 보면

**가능한 것 (2030s~2040s):**
- DUV(28nm) 우주용 특수 칩 (방사선 차폐 프로세서, 센서, MEMS)
- 이 칩들은 9N 순도가 필요 없고, 공정이 단순하며, 수요가 우주에 있다
- "지구에서 칩을 만들어 달로 보내는" 비용 vs "달에서 직접 만드는" 비용의 교차점

**불가능한 것 (50~100년):**
- 최첨단 EUV 2nm 팹: 전력, 순도, 공급망 모두 불가
- 학술 문헌에 달 IC 팹에 대한 진지한 공학 제안 자체가 없음

## 7. 태양전지 — 달에서 "자기 전력"을 만드는 첫 공장

### 왜 태양전지가 먼저인가

MRE(용융 레골리스 전해)로 추출한 5N 실리콘:
- IC 칩에는 못 쓴다 (9N 필요)
- **태양전지에는 충분하다** (5N이면 됨)
- 원료(레골리스)가 발밑에 무한하다

### Blue Origin "Blue Alchemist"

- 레골리스에서 직접 태양전지 + 전선 제조
- NASA $35M 계약
- **2026년 자율 제조 시연 예정**
- 목표: 달에서 "자기 전력원을 스스로 만드는" 자기복제 시스템

### 달 태양광 발전소 현실성

| 방식 | 현실성 | 시기 |
|------|--------|------|
| 수입 패널 (지구에서 가져감) | **7/10** | 5~10년 |
| 현지 제조 (레골리스 → 태양전지) | **5/10** | 2030s~2040s |

**Shackleton Crater 림**: 92~95% 일조율, 3,881 kWh/m²/년 (지구 최적지의 1.8배).

### 논리 흐름

```
태양전지 (5N 실리콘) → 전력 확보 → ISRU (물, 가스) →
→ 팹 기초 인프라 → DUV 팹 운용 → 우주용 칩 생산
```

**달의 첫 공장은 반도체 팹이 아니라 태양전지 공장이다.**
그리고 그 태양전지 공장이 반도체 팹의 전제 조건을 만든다.

## 8. 운송 계산 — "옮길 수 있는가?"

### 장비 무게

| 장비 | 무게 |
|------|------|
| ASML EUV (NXE:3600D) | 180톤 (40개 컨테이너 + 747 3대) |
| ASML High-NA EUV (EXE:5000) | 150톤 (250개 크레이트) |
| ASML DUV | 25~35톤 |
| 에칭/CVD/PVD | 각 8~15톤 |
| 이온 주입기 | 10~20톤 |

### 팹 규모별 총 중량

| 규모 | 장비 수 | 총 중량 |
|------|---------|---------|
| 풀스케일 (TSMC급) | ~1,200개 | ~20,000톤 |
| 1/10 규모 | ~120개 | ~2,000톤 |
| **최소 DUV 팹** | **~20개** | **~500톤** |

### Starship 발사 횟수

최소 DUV 팹 (500톤):

| | 보수적 (15t/착륙) | 낙관적 (50t/착륙) |
|---|---|---|
| 화물 착륙 | 34회 | **10회** |
| 탱커 비행 | 340~544회 | 40~80회 |
| **총 발사** | ~500회 | **~70회** |

Falcon 9는 2024년 한 해에 96회 발사했다. SpaceX가 이미 하고 있는 규모.

### 비용 비교

| 시나리오 | 팹 운송비 | 비교 |
|----------|----------|------|
| Starship 초기 ($100M/발사, 보수적) | ~$500억 | TSMC AZ ($650억)과 비슷 |
| **Starship 성숙기 ($15M/발사, 낙관적)** | **~$10억** | **TSMC AZ의 1/65** |
| Starship 장기 ($5M/발사) | ~$3.5억 | TSMC AZ의 1/185 |

### 연간 소모품 보급

| 구분 | 연간 필요량 | 조달 방식 |
|------|-----------|-----------|
| 물 (UPW) | ~7만~28만 톤 | **달 극지 얼음 (ISRU)** — 지구 수송 불가 |
| 질소 | ~5,000~10,000톤 | **ISRU** — 지구 수송 불가 |
| 특수 가스 | ~10~20톤 | 지구 수송 (연 1회) |
| 포토레지스트 + 화학물질 | ~20~50톤 | 지구 수송 (연 1회) |
| 예비 부품 + 웨이퍼 | ~20~50톤 | 지구 수송 (연 1회) |
| **지구 수송 합계** | **~60~170톤/년** | **Starship 2~4회/년** |

**물과 가스는 지구에서 보낼 수 없다 — 달에서 만들어야 한다.**
이것이 ISRU와 태양광 발전이 반도체 팹보다 선행돼야 하는 이유.

## 9. 선후관계 — 단계별 타임라인

```
1단계 (2028~2030): Starship 완전 재사용 → $100/kg
2단계 (2030~2032): 달 극지 태양광 설치 → 전력 확보
3단계 (2032~2035): ISRU — 얼음에서 물, 레골리스에서 태양전지
4단계 (2035~2040): 최소 로봇 팹 수송 → DUV 28nm 우주용 칩 생산
5단계 (2040~2050+): 규모 확장, EUV 도입 검토
```

**모든 단계의 전제조건이 Starship이다.**

## 10. 아티클 서사 — 제1원칙 패턴

```
통념: "달에서 반도체를 만든다고? 비현실적이다."

분해: 팹은 왜 이렇게 큰가?
  → 오염의 75~80%는 인간이다
  → 전력의 30~50%는 HVAC다
  → 건물의 83%는 클린룸이 아니다

인간을 빼면?
  → 로봇은 진공에서 작동한다 (이미 검증됨)
  → 달의 진공은 ISO 1보다 10조 배 깨끗하다
  → 팹 크기 1/10~1/100, 전력 1/15

Gap: 장벽은 물리학이 아니라 물류다.

가능성: Starship이 $100/kg을 실현하면?
  → 500톤 팹 운송 = Starship 70회 = ~$10억
  → TSMC 애리조나 $650억의 1/65

그리고 모든 길은 Starship으로 통한다.
```
