# 이란 전쟁 지정학 온톨로지 리서치

> 5-8 | 2026-03-28 시작 | 지속 추적형 리서치
> 목적: 이란 전쟁의 나라별 연관관계, 역사, 경제적 영향을 온톨로지로 구조화하여 지속 추적

## 파일 구조

```
iran_conflict/
├── README.md                    ← 이 파일
├── ontology_map.md              ← 나라별 연관관계 + 역할 매핑
├── timeline.md                  ← 전쟁 타임라인 (1979~현재)
├── economic_impact.md           ← 한국/글로벌 경제 영향
├── proxy_network.md             ← 이란 프록시 네트워크 (저항의 축)
└── nuclear_program.md           ← 핵 프로그램 히스토리
```

## 추적 원칙

- **무한 확장형**: 새 이벤트 발생 시 타임라인에 추가
- **나라별 연관관계**: 각 국가의 입장/이해관계/행동을 분리 추적
- **온톨로지 연동**: `config/ontology_config.yaml`의 스토리 클러스터와 연결
- **투자 연결**: 경제적 영향 → 시그널 리포트 → 아티클 테제로 이어지는 흐름
