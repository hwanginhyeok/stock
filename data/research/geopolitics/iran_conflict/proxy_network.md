
## 1. 🇱🇧 헤즈볼라 (레바논)

| 항목 | 상세 |
|------|------|
| **관계 수준** | 이란과 가장 깊은 통합. 사실상 IRGC 해외 지부 |
| **전력** | 로켓/미사일 150,000+ 발, 전투원 30,000~50,000명 |
| **능력** | 정밀유도무기(PGM) 보유 — 이스라엘 전역 타격 가능 |
| **자금** | 이란에서 연 **$7억** 지원 (FDD 추정) |
| **무기 생산** | 레바논 지하시설에서 정밀유도로켓 자체 생산 가능 |
| **2024 상태** | 이스라엘과 전쟁 → 휴전 (지도부 대거 제거됨) |
| **2026 상태** | 휴전 파기, 이스라엘에 로켓 재발사 |
| **약화 요인** | 2024 전쟁으로 지도부 타격, 이란 통신 인프라 파괴로 지휘 체계 단절 |

## 2. 🇾🇪 후티 (안사르 알라)

| 항목 | 상세 |
|------|------|
| **관계 수준** | 이란 지원 받지만 독자적 의사결정 능력 보유 |
| **전력** | 탄도미사일 + 대함미사일 + 드론 (이란 기술 기반) |
| **능력** | 홍해 해상교통 교란, 사우디 인프라 타격, 이스라엘 장거리 미사일 |
| **자금** | 이란에서 연 **$1~2억** 지원 |
| **2024 상태** | 홍해 선박 공격 → 글로벌 해운 교란 |
| **2026 상태** | 3/28 이스라엘 향 탄도미사일 발사 — 공식 참전 |
| **위협** | 호르무즈 해협과 별도로 **홍해/바브엘만데브 해협** 통제 가능 |
| **와일드카드** | 가장 예측 불가능한 프록시. 이란 통제 밖에서 독자 행동 가능 |

## 3. 🇮🇶 시아파 민병대 (PMF / 인민동원군)

| 항목 | 상세 |
|------|------|
| **관계 수준** | IRGC 쿠드스군 직접 지휘 (카타이브 헤즈볼라, 아사이브 아흘 알하크 등) |
| **전력** | 이라크 내 10만+ 전투원, 무장 민병대 연합체 |
| **능력** | 이라크 내 미군 기지 로켓/드론 공격 |
| **역할** | 이라크→시리아→레바논 보급로 확보 |
| **2026 상태** | 미군 기지 반복 공격 중 |
| **제약** | 이라크 정부와의 관계 — 공식 군 편입 vs 민병대 이중 정체성 |

## 6. 2026 전쟁에서의 프록시 상태

| 프록시 | 전투력 | 이란 통제 | 독자 행동 | 현재 역할 |
|--------|--------|----------|----------|----------|
| 헤즈볼라 | 🟡 약화됨 | 🔴 통신 단절 | 🟡 중간 | 이스라엘 북부 전선 견제 |
| 후티 | 🟢 건재 | 🟡 부분 통제 | 🔴 높음 | 이스라엘 미사일 공격 + 해상 위협 |

### 핵심 변화: "통제의 상실"

이스라엘의 이란 통신 인프라 타격으로 IRGC가 프록시들과 암호화 통신을 잃었다. **31명의 지역 IRGC 사령관이 사실상 독자적으로 작전 중.** 이란이 "전쟁을 멈추라"고 해도 프록시들이 따를지 불확실 — 이건 휴전 협상의 가장 큰 장애물.
