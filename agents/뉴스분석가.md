# 뉴스분석가 (News Analyst)

## 역할
한국 및 미국 주식 시장 관련 뉴스를 수집하고, 분류/감성분석/요약을 수행하는 에이전트

## 담당 영역

### 뉴스 수집
- **한국**: 네이버 금융뉴스, 한국경제, 매일경제, 연합뉴스
- **미국**: Reuters, Yahoo Finance, Bloomberg, CNBC
- **수집 방식**: RSS 피드 + 웹 스크래핑 (robots.txt 준수)
- **수집 주기**: 장중 30분 간격, 장외 2시간 간격

### 뉴스 분류
- **카테고리**: 시황, 종목, 산업/섹터, 정책/규제, 글로벌, 실적, IPO, M&A
- **중요도**: 상(시장 전체 영향), 중(섹터/종목 영향), 하(참고)
- **시장**: 한국(KOSPI/KOSDAQ), 미국(NYSE/NASDAQ), 글로벌

### 감성 분석
- 뉴스 제목 + 본문 기반 감성 점수 산출 (-1.0 ~ +1.0)
- Claude API로 문맥 기반 감성 분석 수행
- 단순 키워드 매칭이 아닌 금융 도메인 특화 분석

### 요약 및 태깅
- 뉴스 본문 3줄 요약 생성
- 관련 종목 자동 태깅 (종목명/종목코드 매핑)
- 관련 섹터/테마 태깅

### 중복 필터링
- 제목 유사도 기반 중복 뉴스 필터링 (임계값: 0.85)
- 동일 이벤트에 대한 다수 기사 → 대표 기사 1건 선정
- 24시간 내 중복 체크

## 출력 데이터 모델
```
NewsItem:
  - id: str
  - title: str
  - content: str
  - summary: str (3줄 요약)
  - source: str
  - url: str
  - published_at: datetime
  - category: str
  - importance: str (high/medium/low)
  - sentiment_score: float
  - related_tickers: list[str]
  - related_sectors: list[str]
  - market: str (KR/US/GLOBAL)
```

## 사용 라이브러리
- `feedparser` - RSS 피드 파싱
- `beautifulsoup4` - HTML 파싱
- `requests` - HTTP 요청
- `anthropic` - 감성 분석/요약 (Claude Haiku)

## 연동
- → **시장분석가**: 뉴스 감성 데이터 제공 (시장 심리 지표)
- → **아티클작성가**: 뉴스 요약 데이터 제공 (브리핑/리뷰 소재)
- → **리서치어시스턴트**: 특정 종목/섹터 뉴스 조회
