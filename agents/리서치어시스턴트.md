# 리서치어시스턴트 (Research Assistant)

## 역할
온디맨드 심층 리서치를 수행하는 에이전트. 특정 종목, 섹터, 테마, 크로스마켓 분석을 다른 에이전트의 데이터를 종합하여 제공

## 담당 영역

### 리서치 유형

#### 1. 종목 심층 분석
- 기업 개요 및 사업 구조
- 최근 3~5년 재무 추이 분석
- 기술적 분석 (다중 타임프레임: 일봉/주봉/월봉)
- 최근 뉴스/공시 종합
- 동종업계 비교 (Peer Comparison)
- SWOT 분석

#### 2. 섹터 분석
- 섹터 내 주요 기업 성과 비교
- 섹터 성장 트렌드
- 정책/규제 영향 분석
- 글로벌 섹터 동향 비교 (한국 vs 미국)

#### 3. 테마 분석
- 투자 테마 정의 및 관련 종목 매핑
- 테마 모멘텀 분석 (뉴스 빈도, 검색 트렌드)
- 테마 수혜/피해 종목 분류
- 테마 지속성 평가

#### 4. 크로스마켓 분석
- 한국-미국 시장 상관관계
- 글로벌 매크로 지표 영향 (금리, 환율, 원자재)
- ADR/원주 괴리율 분석
- 글로벌 자금 흐름 추적

### 분석 프레임워크

#### SWOT 분석
- **Strengths**: 기업/섹터 강점
- **Weaknesses**: 약점 및 리스크 요소
- **Opportunities**: 성장 기회
- **Threats**: 외부 위협 요인

#### 리스크 식별
- 시장 리스크 (변동성, 유동성)
- 기업 리스크 (실적, 지배구조)
- 규제 리스크 (정책 변화)
- 글로벌 리스크 (지정학, 환율)

### 데이터 종합 활용
- 뉴스분석가 → 관련 뉴스 히스토리 조회
- 시장분석가 → 가격/지표 데이터 조회
- 과거 아티클/리서치 결과 → 컨텍스트 참조

## 출력 데이터 모델
```
ResearchReport:
  - id: str
  - type: str (stock/sector/theme/cross_market)
  - subject: str (종목명/섹터명/테마명)
  - title: str
  - executive_summary: str
  - sections: list[ReportSection]
  - related_tickers: list[str]
  - risk_factors: list[str]
  - swot: dict | None
  - data_sources: list[str]
  - model_used: str
  - created_at: datetime
  - requested_by: str

ReportSection:
  - heading: str
  - content: str
  - charts: list[str] (차트 이미지 경로)
  - data_tables: list[dict]
```

## 사용 라이브러리
- `anthropic` - Claude API (심층 분석용, Opus 모델)
- `pandas` - 데이터 종합/비교
- `plotly` / `matplotlib` - 리서치 차트 생성

## 연동
- ← **뉴스분석가**: 뉴스 히스토리 데이터
- ← **시장분석가**: 가격/기술적/기본적 분석 데이터
- → **아티클작성가**: 리서치 결과 기반 콘텐츠 생성 요청
