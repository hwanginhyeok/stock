# 4-14: 일론머스크 생태계 상상콘텐츠 — 룰렛 웹앱

> **목적**: 일론머스크 자회사 6개 중 2개를 랜덤으로 뽑는 웹 룰렛. X에 링크 공유 → 누구나 브라우저에서 돌릴 수 있음.
> **산출물**: 단일 웹페이지 (HTML + CSS + JS)
> **호스팅**: GitHub Pages / Vercel / Netlify 중 택 1

---

## 콘셉트

```
[X에 링크 공유] → 클릭 → [웹 룰렛 페이지]
                              ↓
                         [SPIN 버튼]
                              ↓
                    [룰렛 애니메이션 → 2사 뽑힘]
                              ↓
                    [조합 결과 + 상상 질문]
                              ↓
                    [X에 공유하기 버튼] → 바이럴 루프
```

**핵심 루프**: 돌린 사람이 결과를 X에 공유 → 새로운 사람이 링크 클릭 → 또 돌림 → 공유

**조합 수**: C(6,2) = **15가지**

## 기능 요구사항

### F1. 룰렛 UI

- **슬롯머신형 2열** 추천 (2칸이 독립적으로 돌아감)
  - 왼쪽 슬롯: 첫 번째 회사
  - 오른쪽 슬롯: 두 번째 회사
  - 같은 회사가 2번 나오지 않도록 중복 제거
- 또는 **원형 휠** 2번 돌리기 (첫 번째 → 두 번째)
- 애니메이션: 스피닝 → 감속 → 멈춤 (easing: ease-out 또는 cubic-bezier)
- 스핀 소요 시간: 2~3초

### F2. SPIN 버튼

- 큰 버튼, 중앙 배치
- 스피닝 중에는 비활성화 (중복 클릭 방지)
- 결과 나온 후 "다시 돌리기" 가능

### F3. 결과 화면

- 뽑힌 2사 이름 + 아이콘/로고 표시
- 중앙에 "**×**" 또는 "**+**" 기호로 조합 표현
- 하단에 상상 질문 표시:
  - "**{회사A} × {회사B}** — 이 둘이 만나면 어떤 세상이 올까?"
- (선택) 15가지 조합별 미리 작성된 한 줄 힌트 표시

### F4. X 공유 버튼

- 결과 화면에 "X에 공유하기" 버튼
- 클릭 시 X intent URL로 이동:
  ```
  https://twitter.com/intent/tweet?text={인코딩된 텍스트}&url={페이지URL}
  ```
- 공유 텍스트 예시:
  ```
  🎰 일론머스크 생태계 룰렛

  나는 [SpaceX] × [Neuralink] 조합이 나왔다!

  이 둘이 만나면 어떤 세상이 올까?

  👉 너도 돌려봐: {URL}
  ```

### F5. OG 메타태그 (X 카드)

- X에 링크 공유했을 때 카드 미리보기가 떠야 함
- `og:title`: "일론머스크 생태계 룰렛 🎰"
- `og:description`: "Tesla, SpaceX, Neuralink, xAI... 2개를 뽑아 상상하라"
- `og:image`: 대표 이미지 (룰렛 스크린샷 또는 제작 이미지)
- `twitter:card`: `summary_large_image`

### F6. 반응형

- 모바일 우선 (X에서 클릭하면 대부분 모바일)
- 최소 너비: 320px
- 데스크톱에서도 정상 동작
